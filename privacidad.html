Politica de Privacidad y Cookies

Ultima actualizacion: febrero 2026

2. Datos recopilados

Este sitio web no recopila datos personales directamente. La informacion que almacenamos en tu navegador es:

  • Preferencia de idioma (localStorage: menorca_rugby_lang)
  • Estado del consentimiento de cookies (localStorage: mrc_cookie_consent)

Estos datos se almacenan unicamente en tu dispositivo y no se envian a ningun servidor.

3. Servicios de terceros

Este sitio utiliza los siguientes servicios externos que pueden establecer sus propias cookies o recopilar datos:

  • Google Maps (iframe): para mostrar la ubicacion del campo de rugby. Politica de privacidad de Google
  • Google Fonts: para cargar tipografias. Politica de privacidad de Google Fonts
  • Font Awesome (cdnjs.cloudflare.com): para iconos. Politica de privacidad de Cloudflare
  • flagcdn.com: para imagenes de banderas en el selector de idiomas

4. Cookies

Este sitio web no utiliza cookies propias. Utilizamos localStorage del navegador para almacenar tu preferencia de idioma y el estado de este aviso. Los servicios de terceros mencionados anteriormente pueden establecer sus propias cookies segun sus respectivas politicas de privacidad.

5. Base legal

El tratamiento de datos se basa en el consentimiento del usuario (art. 6.1.a RGPD) y el interes legitimo del responsable (art. 6.1.f RGPD) para el correcto funcionamiento del sitio web.

6. Derechos del usuario

De acuerdo con el Reglamento General de Proteccion de Datos (RGPD), tienes derecho a:

  • Acceso a tus datos personales
  • Rectificacion de datos inexactos
  • Supresion de tus datos
  • Portabilidad de tus datos
  • Oposicion al tratamiento
  • Limitacion del tratamiento

Para ejercer estos derechos, contacta con nosotros en info@menorcarugbyclub.com. Tambien puedes presentar una reclamacion ante la Agencia Espanola de Proteccion de Datos (AEPD) en www.aepd.es.

7. Contacto

Para cualquier consulta sobre esta politica de privacidad, puedes escribirnos a info@menorcarugbyclub.com.
